# United States amateur-radio examples

These examples are manually selected starting points for US operators. They are not automatically applied and are not a complete band plan.

To use one:

1. Copy the selected CSV into a new working directory and rename it `channels.csv`.
2. Edit names, frequencies, tones, color code, time slot, contacts, groups, and zones as appropriate.
3. For repeater or DMR examples, replace every placeholder with information independently obtained for the intended system.
4. Import the working directory and review all validation output before saving or writing a codeplug.

The example repeater and DMR frequencies are illustrative only. They do not assert coordination, access permission, local suitability, or transmit authorization.
